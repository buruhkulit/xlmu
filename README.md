# Cara menjalankan dengan TERMUX
1. Update & Upgrade Termux
```
apt update && apt full-upgrade -y
```
2. Install Git
```
pkg install git -y
```
3. Install Python
```
pkg install python
```
4. Install Python Pillow
```
apt install python-pillow
```
5. Clone this repo
```
git clone https://github.com/buruhkulit/xlmu
```
6. Open the folder
```
cd xlmu
```
7. Setup
```
bash setup.sh
```
8. Install Requirements
```
pip install -r requirements.txt
```

# Menambahkan Environment Variables

9. Buat file baru
